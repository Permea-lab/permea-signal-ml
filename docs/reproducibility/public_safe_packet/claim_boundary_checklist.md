# Claim Boundary Checklist

## Allowed Claims

- [ ] Initial computational evidence.
- [ ] Aggregate benchmark-label evaluation.
- [ ] Sequence-derived feature and representation comparison.
- [ ] Candidate prioritization before experimental validation.
- [ ] Public-safe aggregate artifact validation.
- [ ] Bounded reproducibility of public aggregate artifacts.

## Forbidden Claims

- [ ] public dataset availability.
- [ ] independent validation.
- [ ] experimental validation.
- [ ] Wet-lab validation.
- [ ] In-vivo validation.
- [ ] Clinical validation.
- [ ] Therapeutic efficacy.
- [ ] Solved delivery.
- [ ] Universal delivery.
- [ ] SOTA.
- [ ] Deployment-grade model.
- [ ] Biological transport proof.
- [ ] AlphaFold for Delivery as achieved status.

## Required Framing

Use aggregate, benchmark-first, before-experimental-validation language. The packet can support reproducibility of public aggregate artifacts and boundary checks, not public release of row-level data or biological validation.
