## Claim-Boundary Checklist

Before any public use, confirm the package does not claim:

- leakage-free status
- leakage fully controlled
- robust generalization
- validated BBB delivery
- biological validation
- wet-lab validation
- in-vivo validation
- true BBB performance
- therapeutic efficacy
- clinical efficacy
- external expert review
- peer review
- dataset licensing/provenance closure

Allowed framing:

- computational-only evidence
- benchmark-signal claim under documented data and split limitations
- leakage-aware sensitivity estimate under one grouping strategy
- reproducible evidence package
- pre-experimental candidate-prioritization support
