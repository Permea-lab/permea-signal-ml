# Circulation Guide v0.1

## Purpose

This guide helps internal or trusted external readers review the current BBB-oriented Permea manuscript package efficiently and with the correct evidence boundaries in mind.

## Recommended reading order

1. `PREPRINT_DRAFT_V0_1.md`
2. `INTERNAL_REVIEW_MEMO_V0_1.md`
3. `BIORXIV_SUBMISSION_CHECKLIST_V0_1.md`
4. `DATASET.md`
5. `V0_1_EVIDENCE_PACKAGE.md`
6. `PREPRINT_ASSEMBLY_V0_1.md`
7. `SUPPLEMENTARY_OUTLINE_V0_1.md`

## What this package is trying to show

This package is trying to show a BBB-oriented computational wedge, benchmark-oriented baseline evidence, non-trivial signal above class-prior behavior, candidate-prioritization relevance prior to wet-lab validation, an explicit imported-versus-regenerated distinction, and provenance-aware packaging of the current evidence surface.

## What this package is not claiming

This package is not claiming experimental validation, mechanistic causality, universal BBB prediction, therapeutic performance, or broad delivery generalization.

## Suggested review questions

- Is the paper scope appropriately bounded?
- Are the imported-versus-regenerated distinctions clear enough?
- Does the manuscript overstate what the benchmark supports?
- Is the candidate-prioritization framing convincing?
- Are the limitations candid and sufficient?
- Is the package legible to a biologically literate but computationally skeptical reader?
- Are the figures and tables enough for a first external review round?

## Current package status

The package is suitable for internal circulation and structured external review among trusted readers, but it is not yet final for public submission.
